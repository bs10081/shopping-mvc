使用方法
資料庫設定

修改 backend.php 中以下變數，以反映你的資料庫設定：

php
Copy code
$servername = "your_servername";
$username = "your_username";
$password = "your_password";
$dbname = "your_dbname";
資料庫表結構

確保你的資料庫中有一個名為 businesses 的表，結構如下：

sql
Copy code
CREATE TABLE businesses (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    address VARCHAR(255) NOT NULL
);
新增商家

發送 POST 請求到 backend.php，包含以下資料：

php
Copy code
$_POST['add'] = true;
$_POST['name'] = '商家名稱';
$_POST['address'] = '商家地址';
刪除商家

發送 POST 請求到 backend.php，包含以下資料：

php
Copy code
$_POST['delete'] = true;
$_POST['business_id'] = 商家ID;
修改商家

發送 POST 請求到 backend.php，包含以下資料：

php
Copy code
$_POST['update'] = true;
$_POST['business_id'] = 商家ID;
$_POST['new_name'] = '新商家名稱';
$_POST['new_address'] = '新商家地址';
